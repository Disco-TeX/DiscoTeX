Other symbols that need to be tested more thoroughly are:

A single character as an argument to \textcircled{...} produces “a.” If multiple characters are given, it produces “hello.” If no characters are given, it produces “ .”

\(\mathrm{Disco\TeX}\) creates ligatures between characters such as “fl” or “fi.” Placing this command between these characters separates the ligatures, as in “f⁣l” or “f⁣i.”

In \(\mathrm{\TeX}\) accents can be put on any character. If group is provided as an argument, only the first character should be accented.
