Proofs can be made collapsible with the \dtcollapsible command.

You can make a proof not collapsible with the \dtnocollapse command.

Proofs with the \dtstartcollapsed command will start collapsed. If you provide \dtstartcollapsed you need not also use the command dtcollapsible. Collapsibility will be inferred.
